# Sparse Homebrew Tap

This repo is a [Homebrew Tap](https://docs.brew.sh/Taps), 
containing an (autogenerated) formula for the
[Sparse CLI](https://github.com/sparse-data/sparse-cli).

## Usage

```sh
brew install sparse-data/tap/sparse-cli
```
